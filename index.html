git commands

(1) Configuring git

0.git --version
1.git config --global user.name "name"
2.git config --global user.email "name@examle.com"
3.git config --list

(2) clone & status

4.git clone https//example.com(copy link from github repos)
5.git status
6.cd nameoffolder(example) (cd stands for change directory , use to change or open folder)
7.ls , ls -a
8.clear (use to clear terminal)

(3) add & commit

9.git add filename , git add . (for many files)
10.git commit -m"any type message"
11.git push origin main
12.cd .. (use to out from folder)

(4) init commands

13.mkdir nameoffolder (make new directory/folder)
14.git init (use to add new folder to git repos)
15.git remote add origin https//example.com
16.git remote -v(to check remote)
17.git branch (to check branch)
18.git branch -M newname
19.git checkout branchname (toChangeBranch)
20.git checkout -b newBranchName(change to new branch)
21.git branch -d name (delete branch)
22.git diff nameofBrnch (compare two branch)
23.git merge name (merge branches) , using PR in github (pull request)
24.git pull origin main
25.git reset
26.git reset name
26.git reset HEAD~1
27.git log (to check commits)
28.git reset hash/commit link
29.fork on github
